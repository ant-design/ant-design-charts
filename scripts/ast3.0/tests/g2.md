## 使用方式

配置 `brushAxisHighlight` 交互有两种方式：

第一种，传入 `boolean` 设置是否开启交互。


第二种，传入 [配置项](#配置项) 对交互进行配置。

```js
// API
// 第一种方式
chart
  .interval()
  .style('stroke', 'black')
  .style('strokeWidth', 2)
  .viewStyle('viewFill', 'red')
  .viewStyle('contentFill', 'yellow');

// 第二种方式
chart
  .interval()
  .style({
    stroke: 'black',
    strokeWidth: 2,
  })
  .viewStyle({
    viewFill: 'red',
    contentFill: 'yellow',
  });
```
