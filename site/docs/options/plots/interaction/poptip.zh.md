提示缩略文本。

## 开始使用

<img alt="example" src="https://mdn.alipayobjects.com/huamei_qa8qxu/afts/img/A*wAQiRpx1jcMAAAAAAAAAAAAADmJ7AQ/original">

```ts
{
  interaction: {
    poptip: true
  }
}
```

## 选项

| 属性               | 描述         | 类型         | 默认值 |
| ------------------ | ------------ | ------------ | ------ |
| offsetX            | x 方向偏移量 | `number`     | 8      |
| offsetY            | y 方向偏移量 | `number`     | 8      |
| `tip${StyleAttrs}` | poptip 样式  | `StyleAttrs` | -      |
