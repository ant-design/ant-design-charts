#### markBackground

<description>**optional** *BarOptions*</description>

mark 背景配置

```ts
{
  scale: {
    y: {
      domain: [0, 200]
    }
  },
  markBackground: {
    style: {
      fill: '#eee'
    }
  },
}
```
