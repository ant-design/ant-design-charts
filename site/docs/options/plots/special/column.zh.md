#### markBackground

<description>**optional** *ColumnOptions*</description>

mark 背景配置

```ts
{
  scale: {
    y: {
      domain: [0, 0.5]
    }
  },
  markBackground: {
    style: {
      fill: '#eee'
    }
  },
}
```
