`morphing` is a morph animation between shapes, created through transitions between SVG Paths.

## Getting Started

<img alt="morphing" src="https://gw.alipayobjects.com/zos/raptor/1670815385405/animation.gif" width="400" />

```js
{
  "animate": {
    "enter": {
      "type": "morphing"
    }
  }
}
```
