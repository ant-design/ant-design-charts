`growInY` is a matrix animation that enlarges the container along the y direction, mainly used for G.Group container class animations. It affects the `transform.scale` property of the shape.

## Getting Started

<img alt="growInY" src="https://gw.alipayobjects.com/mdn/rms_f5c722/afts/img/A*L6mkQa3aG64AAAAAAAAAAABkARQnAQ" width="400" />

```js
{
  "animate": {
    "enter": {
      "type": "growInY"
    }
  }
}
```
