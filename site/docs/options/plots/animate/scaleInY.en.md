`scaleInY` is a growth animation of a single shape along the y direction. It affects the `transform.scale` property.

## Getting Started

<img alt="scaleInY" src="https://gw.alipayobjects.com/mdn/rms_f5c722/afts/img/A*L6mkQa3aG64AAAAAAAAAAABkARQnAQ" width="400" />

```js
{
  "animate": {
    "enter": {
      "type": "scaleInY"
    }
  }
}
```
