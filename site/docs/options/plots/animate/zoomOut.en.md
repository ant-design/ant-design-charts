`zoomOut` is a zoom-out animation along the center point of the shape. It affects the `transform.scale` property.

## Getting Started

<img alt="zoomOut" src="https://gw.alipayobjects.com/mdn/rms_f5c722/afts/img/A*PZ2gTrkV29YAAAAAAAAAAABkARQnAQ" width="400" />

```js
{
  "animate": {
    "exit": {
      "type": "zoomOut"
    }
  }
}
```
