`zoomIn` is a zoom-in animation along the center point of the shape. It affects the `transform.scale` property.

## Getting Started

<img alt="zoomIn" src="https://gw.alipayobjects.com/mdn/rms_f5c722/afts/img/A*wc4dQp4E6vkAAAAAAAAAAABkARQnAQ" width="400" />

```js
{
  "animate": {
    "enter": {
      "type": "zoomIn"
    }
  }
}
```
