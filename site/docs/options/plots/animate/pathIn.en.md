`pathIn` is an SVG Path entrance animation. Implemented by affecting the `lineDash` property.

## Getting Started

<img alt="pathIn" src="https://gw.alipayobjects.com/mdn/rms_f5c722/afts/img/A*gxZ1RIIMtdIAAAAAAAAAAABkARQnAQ" width="400" />

```js
{
  "animate": {
    "enter": {
      "type": "pathIn"
    }
  }
}
```
