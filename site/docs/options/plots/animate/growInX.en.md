`growInX` is a matrix animation that enlarges the container along the x direction, mainly used for G.Group container class animations. It affects the `transform.scale` property of the shape.

## Getting Started

<img alt="growInX" src="https://gw.alipayobjects.com/mdn/rms_f5c722/afts/img/A*vhRVSLxDqU8AAAAAAAAAAABkARQnAQ" width="400" />

```js
{
  "animate": {
    "enter": {
      "type": "growInX"
    }
  }
}
```
