`scaleInX` is a growth animation of a single shape along the x direction. It affects the `transform.scale` property.

## Getting Started

<img alt="scaleInX" src="https://gw.alipayobjects.com/mdn/rms_f5c722/afts/img/A*oiaGTLx-dNcAAAAAAAAAAABkARQnAQ" width="400" />

```js
{
  "animate": {
    "enter": {
      "type": "scaleInX"
    }
  }
}
```
