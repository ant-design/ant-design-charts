`scaleOutX` is a disappearing animation of a single shape along the x direction. It affects the `transform.scale` property.

## Getting Started

<img alt="scaleOutX" src="https://mdn.alipayobjects.com/huamei_qa8qxu/afts/img/A*XpI1SbHQADUAAAAAAAAAAAAADmJ7AQ/original" width="400" />

```js
{
  "animate": {
    "exit": {
      "type": "scaleOutX"
    }
  }
}
```
