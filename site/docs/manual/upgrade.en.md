## Encountering Problems

We have tried to collect all known incompatible changes and their impacts, but it's possible that there are still some scenarios we haven't considered. If you encounter problems during the upgrade process, please provide feedback on [GitHub issues](https://github.com/ant-design/ant-design-charts/issues). We will respond as soon as possible and improve this document accordingly.
