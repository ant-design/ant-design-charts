order: 1

## `zoom.`position

`string` optional default: `'topleft'`

控件的位置，支持以下定位位置：

*   'topright'
*   'topleft'
*   'bottomright'
*   'bottomleft'

## `zoom.`zoomInText

`string` optional default: `'+'`

放大按钮文本。

## `zoom.`zoomInTitle

`string` optional default: `Zoom in`

放大按钮名称。

## `zoom.`zoomOutText

`string` optional default: `'&#x2212;'`

缩小按钮文本。

## `zoom.`zoomOutTitle

`string` optional default: `Zoom out`

缩小按钮名称。
