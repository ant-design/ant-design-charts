### `options.`animate

`boolean｜AnimateAttr` optional

水波动画，AnimateAttr 配置如下：

| 属性   | 描述         | 类型      | 默认值  | 是否必填 |
| ------ | ------------ | --------- | ------- | -------- |
| enable | 是否开启动画 | `boolean` | `false` | optional |
| speed  | 水波速度     | `number`  |         | optional |
| rings  | 水波环数     | `number`  |         | optional |
