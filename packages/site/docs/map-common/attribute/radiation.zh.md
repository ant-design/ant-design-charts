### `options.`radiation

`object` optional

落地点辐射圈。

```js
{
  radiation: {
    color: 'yellow',
    size: 20,
  }
}
```

#### `radiation.`enabled

`boolean` optional `true`

是否开启辐射圈。

#### `radiation.`color

`string` optional

辐射圈颜色。

#### `radiation.`size

`number|object|Function` optional default: `20`

辐射圈大小。

#### `radiation.`animate

`boolean` optional `true`

是否启用辐射圈动画。
